## 1. 기획의도

현재 기업체에서 홍보 목적으로 개설한 모임 사이트는 다수 존재한다.
광고 기능을 배제하고, 순수 사람들 간의 취미 혹은 작은 목표를 공유하여 소소한 성취감을 느낄 수 있는 사이트를 제작하고자 한다.
